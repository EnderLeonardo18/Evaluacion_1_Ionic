# Aplicación Ionic con Angular (Standalone Components)

Este proyecto es una aplicación móvil híbrida desarrollada con **Ionic Framework** y **Angular**, enfocada en implementar una interfaz moderna con navegación lateral.

## Características Técnicas
* **Arquitectura Standalone**: Desarrollo basado en componentes independientes para un código más modular y eficiente.
* **Navegación Lateral**: Implementación de `ion-split-pane` para soporte responsivo en móviles y escritorio.
* **Estilos Personalizados**: Uso de **SCSS** para la implementación de efectos visuales como *Glassmorphism* y variables de color personalizadas.
* **Carga Dinámica de Iconos**: Optimización de rendimiento mediante el uso selectivo de `ionicons`.

## Estructura del Proyecto
El proyecto se organiza en las siguientes páginas principales creadas vía CLI:
* `inicio`: Pantalla principal de la aplicación.
* `informacion`: Sección con detalles técnicos y académicos.
* `contacto`: Enlace directo a repositorio de código.
