# Aplicación Ionic con Angular (Standalone Components)

Este proyecto es una aplicación móvil híbrida desarrollada con **Ionic Framework** y **Angular**, enfocada en implementar una navegación fluida y una identidad visual personalizada.

## Características Técnicas
* **Arquitectura Standalone**: Desarrollo basado en componentes independientes para un código más modular y eficiente.
* **Navegación Lateral Responsiva**: Uso de `ion-split-pane` para adaptarse a diferentes tamaños de pantalla.
* **Estilos con SCSS**: Implementación de una temática personalizada basada en colores sólidos y degradados, alejándome de los estilos por defecto de Ionic.
* **Gestión de Iconos**: Carga dinámica de `ionicons` para optimizar el rendimiento de la app.

## Identidad Visual y Temática
A diferencia de los diseños estándar, decidí darle un toque personal y con más carácter a la interfaz:
* **Paleta de Colores**: Inspirada en un estilo elegante y deportivo, utilizando un **Azul Noche** para el fondo y un **Rojo Vinotinto/Carmesí** para los elementos principales.
* **Temática**: La sección de Inicio está dedicada al **Clásico Mundial de Béisbol**, utilizando estos colores para celebrar la identidad de Venezuela de una forma profesional y limpia.

## Estructura de Vistas
* `inicio`: Pantalla de bienvenida con la temática deportiva.
* `informacion`: Detalle de mi formación académica en la **UNETI** y perfil técnico.
* `contacto`: Enlace directo para comunicación y revisión de código.

## Cómo ejecutar el proyecto
1. Instalar dependencias: `npm install`
2. Correr el servidor local: `ionic serve`
